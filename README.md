# Car-Evaluation-Model
— Built a machine learning model to classify car acceptability using the Kaggle Car Evaluation dataset. Applied data preprocessing, EDA, and compared Decision Tree, Random Forest, and KNN models to achieve high classification accuracy.

INTRODUCTION


This project focuses on predicting car acceptability using machine learning. A dataset from Kaggle is used, which includes features like buying price, maintenance cost, number of doors, passenger capacity, and safety.
The data is first prepared using preprocessing techniques such as label encoding and train-test splitting. Exploratory Data Analysis (EDA) is performed to understand patterns in the data.
Since the data is labeled, supervised learning methods are used. Different algorithms like Decision Tree, Random Forest, and K-Nearest Neighbors are applied and compared to find the best model.
The model’s performance is evaluated using accuracy and  confusion matrix.

PROBLEM STATEMENT


The evaluation of the acceptability of cars is a significant process, and it depends on several factors such as the price of the car, maintenance cost, safety, the number of passengers and etc. Performing this multi-factor analysis manually to determine a car's acceptability is a highly time-consuming process. 
Therefore, the main aim of this project is to address this issue by developing a machine learning model capable of automatically classifying cars into different acceptability categories using the Car Evaluation dataset. 
It also aims to compare the machine learning algorithms to identify the most efficient algorithm.
Additionally, the project also focuses on calculating feature importance to identify which features contribute the most in determining the acceptability of a car.
